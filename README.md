## Features
- Supports (Android & iOS coming soon)
- Create, read, update, and delete notes.
- log in and sign up user
- Labels for notes
- Color tags
- Smooth UI



## This project is done with
- Flutter Framework
- Local Storage(SQLite)
- SharedPreferences
